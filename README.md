# oss-version

Claude Code skill for checking and upgrading locally-installed open-source CLI tools.

## Usage

Trigger phrases:
- "检查组件版本"
- "升级 rtk"
- "列出组件清单"
- "加个组件到清单"

## Structure

- `SKILL.md` — skill description and workflow
- `components.toml` — registry of managed components
- `scripts/oss-version.py` — implementation
